## Setup

1. Go to project folder

> cd backend

2. Install Dependencies

> npm run setup

3. Create .env with following content

```
NODE_ENV=development

DB_HOST=127.0.0.1
DB_DATABASE=exampleDatabase

DB_USER=root
DB_PASSWORD=root

PORT=3333

SECRET=AAAAA
```

## Run

1. Start Server

> npm start

2. Open in url or using an Endpoint tester [Insomnia](https://insomnia.rest/)

> http://127.0.0.1:3333/
